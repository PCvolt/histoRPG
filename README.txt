(no command line is needed)
Upload your work on your branch.

Before uploading/creating a (new) file, you'll be asked :
"
 - Commit directly to the 'master' branch.
 - Create a new branch for this commit and start a pull request. Learn more about pull requests.
"
The name between ' ' refers to the branch you are working on.
Do not commit on the master branch until the file isn't in final version.
Commit on your dedicated branch.


Commits = Save changes (to a file, a new upload, etc.)

Pull Request = Changes are accessible to others. Discuss and review the potential changes and add follow-up commits before the changes are merged into the repository.
(you can mention with @username)
